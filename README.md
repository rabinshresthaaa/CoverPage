# LabCover

LabCover is a simple web application that generates a **Lab Report cover page** in Microsoft Word format.

This application was developed for personal college use, to generate the cover page of my Lab Report quickly and consistently.

## Features
- Enter lab and student details using a simple form
- Generates a formatted `.docx` cover page
- Uses Times New Roman font
- Clean academic layout
- Responsive UI with preview section

## Tech Stack
- ASP.NET Core MVC
- OpenXML SDK
- Tailwind CSS

## Usage
1. Open the application
2. Fill in your lab report details
3. Click **Generate Cover Page**
4. Download the Word document

## Purpose
This project is intended for personal academic and learning purposes.
